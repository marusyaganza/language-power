# language-power

## Purpose

Language power is a simple web app for English learners. It's recommended for B2+ learners, for whom English to English translation is preferable.

## Technological stack

* React
* Webpack
* Babel
* Eslint, Stylelint
* Jest
* Cypress

## How to use it?
